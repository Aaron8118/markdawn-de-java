# Lenguaje de programacion Java

Bienvenido a este documento de mkdocs sobre Java.

---

## ¿Qué es Java?

> Java es un lenguaje de programación y una plataforma tecnológica muy utilizada para desarrollar todo tipo de aplicaciones: desde programas de escritorio hasta apps móviles y sistemas empresariales.

Fue creado en 1995 por Sun Microsystems (luego adquirido por Oracle Corporation) y se diseñó con una idea clave: “escribir una vez, ejecutar en cualquier lugar”.

---

## ¿Como funciona?

Java funciona gracias a la Máquina Virtual de Java (JVM), que permite que el mismo programa se ejecute en diferentes sistemas (Windows, Mac, Linux) sin cambios.

---

## POO

Es un paradigma de programación que organiza el código en objetos, que representan entidades del mundo real.

En POO es una forma de programar donde:

Se crean ***clases*** (*plantillas*). [^1]

Y a partir de ellas se crean **objetos** (*instancias*).

Cada objeto tiene:

- **Atributos** (*datos*).
- **Métodos** (*acciones*).

[^1]: Existren otros elementos ademas de las clases. 

---

## Caracteristicas

- **Encapsulación**: protege los datos de una clase y controla el acceso mediante métodos públicos.
- **Abstracción**: permite representar conceptos del mundo real con clases y objetos sin mostrar detalles complejos.
- **Herencia**: facilita la reutilización de código y la creación de jerarquías entre clases.
- **Polimorfismo**: permite tratar objetos de distintas clases derivadas como si fueran del mismo tipo padre.
- **Modularidad**: organiza el código en unidades independientes y fáciles de mantener.
- **Reutilización**: promueve el uso de componentes ya creados en lugar de escribir código nuevo desde cero.
