# Objetos

Un objeto es una instancia de una clase.

## Crear un objeto

```java
Persona p = new Persona();
p.nombre = "Ana";
p.edad = 25;
p.saludar();
```

## Acceso a miembros

- `p.nombre` accede al atributo
- `p.saludar()` invoca un método

## Concepto

Los objetos representan entidades del mundo real y contienen estado (atributos) y comportamiento (métodos).
